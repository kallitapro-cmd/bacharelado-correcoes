## Contexto

Com o Sprint 0 concluído e todos os ADRs aprovados, o projeto entra na fase de código. Esta story cria a base técnica que todas as demais stories do Sprint 1 precisam: estrutura de pastas, dependências pinadas, configuração do Streamlit e o esqueleto do `app.py`. Sem esta story concluída, nenhuma outra story do Sprint 1 pode ser iniciada.

A stack está definida na arquitetura v2.0: Streamlit + Python 3.11+, anthropic SDK, streamlit-authenticator, pydantic, openpyxl, PyMuPDF (fitz), python-pptx, python-docx, Pillow + pytesseract, streamlit-aggrid, sqlite3 (stdlib), reportlab (apenas dev/test).

## Objetivo

Criar a estrutura inicial do repositório com todas as dependências pinadas, `.gitignore` completo, esqueleto de `app.py`, configuração básica do Streamlit e estrutura de pastas `src/` pronta para uso pelas demais stories do Sprint 1.

- [x] Todas as dependências instalam com `pip install -r requirements.txt` sem conflitos

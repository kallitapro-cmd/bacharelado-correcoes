## Contexto

O ADR-006 (`docs/decisions/ADR-006-plagio-policy.md`) define a política completa de detecção de similaridade textual do Corretor Acadêmico. Em turmas de até 120 alunos, há risco real de cópia direta entre trabalhos entregues individualmente. O sistema deve sinalizar pares suspeitos ao Professor Assistente (PA) sem jamais alterar notas automaticamente — a decisão pedagógica é 100% do PA.

Esta story implementa o módulo de detecção de similaridade como **componente puro e isolado**: recebe uma lista de `Trabalho`, compara todos os pares O(n²) via `difflib.SequenceMatcher` (stdlib Python, zero dependências externas), classifica cada par por severidade e retorna a lista de pares suspeitos. O módulo **não tem acesso ao campo `nota`** — separação estrutural conforme ADR-006, seção "Garantia Explícita".

O processamento ocorre 100% localmente, sem envio de qualquer texto para serviços externos — conforme ADR-004 (retenção zero / privacidade LGPD).

### Estrutura de dados de entrada

O detector recebe uma lista de objetos com `aluno_id` (str) e `texto` (str). O campo `nota` **não deve existir** na interface de entrada do detector — usar uma dataclass de input separada ou Typed-Subset, não o tipo `Trabalho` completo com nota. Isso garante a separação estrutural exigida pelo AC-06.

Sugestão de interface mínima:

```python
from dataclasses import dataclass

@dataclass
class TrabalhoParaComparacao:
    aluno_id: str   # RA como string
    texto: str      # conteúdo textual extraído do trabalho
    # SEM campo nota — separação estrutural conforme ADR-006
```

### Assinaturas exatas das funções públicas

Conforme ADR-006, as assinaturas devem ser:

```python
def calcular_similaridade(texto_a: str, texto_b: str) -> float:
    """Retorna razão de similaridade entre 0.0 e 1.0."""
    ...

def preprocess(texto: str) -> str:
    """Normaliza whitespace, lowercase, remove não-textuais. NÃO remove stopwords."""
    ...

def detectar_plagio_no_batch(
    trabalhos: list[TrabalhoParaComparacao],
    threshold: float = 0.70,
) -> list[ParPlagio]:
    """Compara todos os pares O(n²). Retorna pares com similaridade >= threshold."""
    ...
```

### Implementação de referência do ADR-006

O ADR-006 fornece pseudo-código completo para todas as funções. O `@dev` deve implementar exatamente o padrão descrito na seção "Algoritmo de Detecção" do ADR-006, incluindo a chamada `difflib.SequenceMatcher(None, texto_a, texto_b).ratio()` sem variações.

### Severidade (tabela exata do ADR-006)

| Similaridade | Severidade |
|-------------|-----------|
| 70% ≤ sim < 90% | `"amarelo"` |
| 90% ≤ sim ≤ 100% | `"vermelho"` |
| < 70% | não incluído no resultado |

Nota: o threshold padrão é 70% (`0.70`). Pares com similaridade exatamente igual ao threshold são incluídos.

### Pré-processamento (regras do ADR-006)

1. Normalizar whitespace — múltiplos espaços/quebras → espaço único
2. Converter para lowercase
3. Remover caracteres não-textuais (números de página, headers/footers detectáveis)
4. **NÃO remover stopwords** — manter viés igual entre textos curtos e longos

### Performance esperada (ADR-006)

Para batch típico de 120 alunos:
- Número de pares: `120 × 119 / 2 = 7.140`
- Tempo médio por comparação: ~3-5ms
- Tempo total estimado: ~25-35s (bem abaixo do limite AC-08 de 60s)

## Riscos

- **Risco 1:** A Story 2.2 (`batch_processor.py`) pode ainda não existir quando esta story for implementada. Mitigação: `plagiarism_detector.py` é isolado — não importa `batch_processor`; a integração entre os dois módulos é responsabilidade do batch_processor (Story 2.2). Esta story pode ser implementada de forma independente.
- **Risco 2:** A definição de `Trabalho` em `batch_processor.py` pode incluir `nota`; importar esse tipo diretamente no detector violaria o AC-06. Mitigação: usar `TrabalhoParaComparacao` como tipo de entrada próprio do detector — não depender do tipo completo de `batch_processor`.
- **Risco 3:** Performance pode variar em ambiente CI com menos CPU. Mitigação: o limite de AC-08 é 60s, bem acima dos 25-35s esperados; margem de 2x é suficiente para variação de ambiente.

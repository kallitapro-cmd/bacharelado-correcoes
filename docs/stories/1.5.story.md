## Contexto

O formato mais comum de entrega de trabalhos acadêmicos no Bacharelado em Administração é PDF. Os PDFs chegam em três variações:
1. PDFs nativos com texto extraível (gerados por Word, LaTeX, Google Docs)
2. PDFs raster/scaneados (sem texto, apenas imagem) — necessitam OCR
3. PDFs corrompidos ou protegidos por senha

O conversor deve tentar primeiro a extração nativa via PyMuPDF (fitz) — mais rápida e fiel — e recorrer ao pytesseract OCR apenas se o texto extraído for insuficiente (menos de 100 caracteres úteis). Arquivos > 10MB são rejeitados antes do processamento para proteger a performance e o custo.

## Objetivo

Implementar `src/converters/pdf_converter.py` com extração de texto de PDFs via PyMuPDF com fallback para OCR pytesseract, sanitização de encoding, detecção de falha e limite de tamanho.

## Notas Técnicas

**Assinatura da função principal:**
```python
def convert_pdf(file_path: str | Path) -> dict:
    """
    Converte um arquivo PDF em texto.

    Returns:
        dict com campos:
        - texto (str): texto extraído ou string vazia se falhou
        - paginas (int): número de páginas
        - metodo_extracao (str): "nativo", "ocr" ou "nenhum"
        - flags (list[str]): ["arquivo_ilegivel"] se falhou completamente,
                             ["extracao_falhou"] se texto insuficiente
    """
```

**Lógica de fallback:**
```python
# Pseudocódigo
texto_nativo = extrair_com_pymupdf(pdf)
if len(texto_nativo.strip()) < 100:
    texto_ocr = extrair_com_ocr(pdf)
    return {texto: texto_ocr, metodo_extracao: "ocr", ...}
return {texto: texto_nativo, metodo_extracao: "nativo", ...}
```

**Limite de tamanho:** verificar `os.path.getsize(file_path)` antes de abrir o arquivo. Limite: `10 * 1024 * 1024` bytes (10MB).

**Flags possíveis:**
- `arquivo_ilegivel`: não foi possível abrir o arquivo (corrompido, senha, formato inválido)
- `extracao_falhou`: arquivo aberto mas nenhum método produziu texto útil
- (nenhuma flag = sucesso)

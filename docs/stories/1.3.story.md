**Status:** Done

**Complexidade:** S
**Prioridade:** Alta
**Dependências:** 1.1, 1.2

## Contexto

O Corretor Acadêmico será hospedado no Streamlit Cloud para que o PA possa acessá-lo de qualquer lugar sem instalação local. O Streamlit Cloud tem requisitos específicos: variáveis de ambiente via `secrets.toml`, dependências de sistema via `packages.txt`, e configuração de autenticação.

Esta story é principalmente documental e de validação: garantir que as configurações do Sprint 1 são suficientes para um deploy funcional, documentar o processo e preparar o exemplo de `secrets.toml` para que o PA possa configurar o ambiente de produção.

## Objetivo

Documentar o processo de deploy completo no Streamlit Cloud, criar o `secrets.toml.example` com todas as variáveis necessárias, e garantir que a app inicializa corretamente (com mensagem útil) quando variáveis obrigatórias estão ausentes.

## Notas Técnicas

**Estrutura do secrets.toml.example:**
```toml
# Chave da API Anthropic (obrigatório)
ANTHROPIC_API_KEY = "sk-ant-..."

# Configurações da aplicação (opcionais com padrão)
MAX_COST_BRL = "15.0"   # Hard limit de custo por sessão (ADR-005)
```

**Checklist de variáveis obrigatórias:**
| Variável | Obrigatório | Padrão | Referência |
|----------|-------------|--------|------------|
| `ANTHROPIC_API_KEY` | Sim | — | ADR-003 |
| `MAX_COST_BRL` | Não | `15.0` | ADR-005 |

**Startup check em app.py:**
```python
import streamlit as st

api_key = st.secrets.get("ANTHROPIC_API_KEY", None)
if not api_key:
    st.error("ANTHROPIC_API_KEY não configurada. Acesse Settings > Secrets no Streamlit Cloud e adicione sua chave.")
    st.stop()
```

**Passos principais do deploy (para docs/guides/deploy.md):**
1. Fork/push do repositório para GitHub
2. Acessar share.streamlit.io → New app
3. Conectar repositório e apontar para `app.py`
4. Em Settings → Secrets: adicionar `ANTHROPIC_API_KEY` e demais variáveis
5. Adicionar `config.yaml` real (credenciais hashed) também em Secrets ou via arquivo não-versionado
6. Clicar Deploy

**Referências:** ADR-003 (modelos), ADR-004 (privacidade), ADR-005 (custo/hard limit).

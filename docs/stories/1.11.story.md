# Story 1.11 — Implementar safe_logger com sanitização de dados sensíveis

**Status:** Done
**Sprint:** 1 — Fundação técnica
**Complexidade:** S
**Prioridade:** Alta
**Dependências:** 1.1, ADR-004

## Contexto

O ADR-004 (LGPD e Retenção Zero) estabelece que o sistema nunca deve logar dados pessoais: RAs de alunos, nomes, notas, conteúdo de trabalhos, e API keys da Anthropic. Erros comuns que um logger padrão deixaria vazar incluem: RA em mensagens de debug, `sk-ant-*` em stack traces, e nomes de alunos em mensagens de exceção.

O `safe_logger` é um wrapper sobre o módulo `logging` padrão do Python que intercepta automaticamente mensagens e aplica regex para censurar dados sensíveis antes de qualquer escrita em log. Também customiza o `sys.excepthook` para interceptar exceções não tratadas e sanitizá-las antes de exibir.

## Objetivo

Implementar `src/utils/logger.py` com um wrapper `safe_logger` que aplica sanitização automática de dados sensíveis (RAs, API keys, nomes de alunos) em todas as mensagens de log, e `sys.excepthook` customizado para interceptar exceções não tratadas.

## Notas Técnicas

**Interface mínima:**

```python
# src/utils/logger.py
import logging
import re
import sys

# Padrões de sanitização
PATTERNS = [
    (re.compile(r'\bsk-ant-[a-zA-Z0-9_-]+'), '[API_KEY_REDACTED]'),
    (re.compile(r'\b\d{11}\b'), '[RA_REDACTED]'),
]

def sanitize(message: str) -> str:
    """Aplica todos os padrões de sanitização a uma mensagem."""
    for pattern, replacement in PATTERNS:
        message = pattern.sub(replacement, message)
    return message

class SanitizingFilter(logging.Filter):
    def filter(self, record: logging.LogRecord) -> bool:
        record.msg = sanitize(str(record.msg))
        if record.args:
            # Sanitizar argumentos de formatação do logging
            if isinstance(record.args, tuple):
                record.args = tuple(sanitize(str(a)) for a in record.args)
            elif isinstance(record.args, dict):
                record.args = {k: sanitize(str(v)) for k, v in record.args.items()}
        return True

def get_safe_logger(name: str) -> logging.Logger:
    """Retorna logger configurado com filtro de sanitização."""
    logger = logging.getLogger(name)
    if not logger.handlers:
        handler = logging.StreamHandler()
        handler.addFilter(SanitizingFilter())
        logger.addHandler(handler)
    return logger

# Logger padrão da aplicação
safe_logger = get_safe_logger('corretor_academico')
```

**Customização do sys.excepthook:**

```python
_original_excepthook = sys.excepthook

def _safe_excepthook(exc_type, exc_value, exc_tb):
    """Intercepta exceções não tratadas e sanitiza antes de exibir."""
    import traceback
    tb_str = ''.join(traceback.format_exception(exc_type, exc_value, exc_tb))
    sanitized = sanitize(tb_str)
    sys.stderr.write(sanitized)

sys.excepthook = _safe_excepthook
```

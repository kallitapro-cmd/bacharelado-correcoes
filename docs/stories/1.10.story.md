## Contexto

A Aba 1 da planilha do PA é o manifesto de alunos: contém RA, Nome Completo, E-mail e Telefone de todos os alunos da turma. Essa planilha é o ponto de entrada do pipeline — sem ela, não há como saber quais alunos existem, quais RAs são válidos e como fazer o matching entre entregas e notas.

O ADR-001 definiu com precisão a função `normalize_ra()` que deve ser usada em toda leitura de RA. Esta story implementa essa função exatamente como especificada no ADR-001 e o leitor da Aba 1 que aplica a normalização a todos os RAs lidos.

A planilha real tem 124 alunos. As fixtures têm 8 alunos anonimizados. O leitor deve ser tolerante a variações de coluna (nome pode ser "Nome Completo", "Nome", "NOME", etc.) e a colunas extras.

## Objective

Implementar `src/matching/ra_normalizer.py` com a função `normalize_ra()` exatamente conforme ADR-001, e `src/excel/excel_reader.py` que lê a Aba 1 da planilha do PA retornando lista de `Aluno` com todos os RAs normalizados.

## Notas Técnicas

**Implementação exata de normalize_ra() conforme ADR-001:**

```python
def normalize_ra(ra: str) -> str:
    """
    Normaliza um Registro Acadêmico (RA) para o formato canônico de 11 dígitos.

    Regra (ADR-001):
    - 11 dígitos          → retorna como está
    - 10 dígitos + 2025/  → insere '0' na posição 4 (após o ano)
      2026 prefix
    - Outros formatos     → retorna como está (Camada 3 — revisão manual)

    IMPORTANTE: NUNCA usar zfill(11) — produz resultado inválido (ADR-001).
    """
    ra = str(ra).strip()
    if len(ra) == 10 and (ra.startswith('2026') or ra.startswith('2025')):
        return ra[:4] + '0' + ra[4:]
    return ra
```

```python
import openpyxl

def read_aba1(file_path: str | Path) -> list[Aluno]:
    """
    Lê a Aba 1 (Alunos) da planilha do PA.
    Detecta colunas por nome, normaliza RAs via normalize_ra().
    """
    wb = openpyxl.load_workbook(file_path, data_only=True)
    ws = wb.worksheets[0]  # Aba 1 é sempre a primeira aba

    # Detectar colunas por nome (linha 1 é o cabeçalho)
    header = {cell.value.strip().lower(): idx
              for idx, cell in enumerate(ws[1]) if cell.value}

    # Mapeamento tolerante (aceita variações de nome de coluna)
    col_ra = _find_col(header, ['ra', 'registro acadêmico', 'código'])
    col_nome = _find_col(header, ['nome completo', 'nome', 'aluno'])
    col_email = _find_col(header, ['e-mail', 'email', 'e_mail'])
    col_telefone = _find_col(header, ['telefone', 'tel', 'celular'])

    alunos = []
    for row in ws.iter_rows(min_row=2, values_only=True):
        ra_bruto = str(row[col_ra]) if row[col_ra] else ""
        if not ra_bruto:
            continue
        ra_normalizado = normalize_ra(ra_bruto)
        alunos.append(Aluno(
            ra=ra_normalizado,
            nome=str(row[col_nome]) if row[col_nome] else "",
            email=str(row[col_email]) if col_email is not None and row[col_email] else None,
            telefone=str(row[col_telefone]) if col_telefone is not None and row[col_telefone] else None,
        ))
    return alunos
```

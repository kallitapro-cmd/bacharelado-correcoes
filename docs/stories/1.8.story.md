# Story 1.8 — Implementar conversor de imagem → texto (Pillow + Tesseract OCR)

**Status:** Done
**Sprint:** 1 — Fundação técnica
**Complexidade:** S
**Prioridade:** Alta
**Dependências:** 1.1

## Contexto

Alguns alunos entregam fotos de trabalhos escritos à mão ou capturas de tela de ferramentas externas (whiteboard, miro, etc.). O conversor de imagem usa pytesseract com Pillow para extrair texto via OCR dessas imagens.

O limite de tamanho é menor que para documentos (5MB vs 10MB) porque imagens muito grandes geralmente indicam fotos em alta resolução desnecessárias para OCR, e o processamento de imagens grandes é significativamente mais lento.

## Objetivo

Implementar `src/converters/image_converter.py` com extração de texto de imagens JPG, PNG e WEBP via pytesseract com language='por', mantendo interface de output compatível com os demais conversores.

## Acceptance Criteria

- [x] JPG ou PNG com texto legível em português converte para string não-vazia
- [x] Imagem com tamanho maior que 5MB é rejeitada com mensagem clara antes de qualquer processamento
- [x] Formato não suportado (ex: `.gif`, `.bmp`) retorna dict com flag `formato_nao_suportado`
- [x] Output é sempre um `dict` com os campos: `texto` (str), `paginas` (int, sempre 1 para imagens), `metodo_extracao` (str: "ocr"), `flags` (list[str])
- [x] Testes unitários passam com `pytest tests/unit/test_image_converter.py`

## Notas Técnicas

**Assinatura da função principal:**
```python
def convert_image(file_path: str | Path) -> dict:
    """
    Converte uma imagem em texto via OCR.

    Returns:
        dict com campos:
        - texto (str): texto extraído via OCR
        - paginas (int): sempre 1 para imagens individuais
        - metodo_extracao (str): "ocr"
        - flags (list[str]): ["arquivo_ilegivel"], ["formato_nao_suportado"] ou []
    """
```

**Formatos suportados:**
```python
FORMATOS_SUPORTADOS = {'.jpg', '.jpeg', '.png', '.webp'}

extensao = Path(file_path).suffix.lower()
if extensao not in FORMATOS_SUPORTADOS:
    return {"texto": "", "paginas": 1, "metodo_extracao": "nenhum",
            "flags": ["formato_nao_suportado"]}
```

**Pré-processamento com Pillow:**
```python
from PIL import Image
import pytesseract

img = Image.open(file_path).convert('RGB')
texto = pytesseract.image_to_string(img, lang='por')
```

**Fixture sintética para testes:** criar imagem PNG com texto legível usando Pillow (`ImageDraw`) no próprio arquivo de teste, sem necessidade de arquivo em disco. Alternativa: usar `reportlab` para gerar um PNG sintético em `docs/fixtures/`.

**Limite de tamanho:** 5MB — `5 * 1024 * 1024` bytes.

**Dependência de sistema:** `tesseract-ocr` e `tesseract-ocr-por` declarados em `packages.txt` (Story 1.1). Em ambiente de desenvolvimento local, o tesseract deve estar instalado via apt/brew.
